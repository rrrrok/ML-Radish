# ML-Radish

무 도매가 예측 머신러닝 프로젝트

## v1 전처리 작업 내용

`make_price_quantity_v1.py` 스크립트는 다음과 같은 전처리 작업을 수행합니다:

### 1. 비어있는 날짜 채우기
- 각 그룹(지역/품종/등급)별로 날짜 범위의 모든 날짜 생성
- 누락된 날짜는 숫자 컬럼을 NaN으로 채워 추가
- 연속적인 시계열 데이터 구성

### 2. 휴일 처리
- `holidays` 라이브러리를 사용하여 한국 공휴일 자동 인식
- 주말 및 공휴일 식별
- 휴일의 결측치는 **선형보간(linear interpolation)**으로 채움

### 3. 결측치 채우기 (다단계 전략)

#### 가격 데이터
1. **전일 데이터로 채우기**: 같은 그룹 내에서 전날 값으로 채우기
2. **전년 동일일자 데이터로 채우기**: 계절성을 고려하여 1년 전 같은 날짜의 값 사용
3. **선형 보간**: 그룹별로 선형 보간 수행
4. **이동평균**: 5일 윈도우 이동평균 사용
5. **요일별 평균**: 같은 요일의 평균값으로 채우기
6. **그룹별 평균**: 같은 그룹의 전체 평균값으로 채우기
7. **전체 평균**: 최후의 수단으로 전체 평균값 사용

#### 반입량 데이터
1. **전일 데이터로 채우기**: 같은 그룹 내에서 전날 값으로 채우기
2. **전년 동일일자 데이터로 채우기**: 계절성을 고려하여 1년 전 같은 날짜의 값 사용
3. **선형 보간**: 그룹별로 선형 보간 수행
4. **이동평균**: 7일 윈도우 이동평균 사용
5. **요일별 평균**: 같은 요일의 평균값으로 채우기
6. **그룹별 평균**: 같은 그룹의 전체 평균값으로 채우기
7. **전체 평균**: 최후의 수단으로 전체 평균값 사용


## 출력 파일

- `price_data_v1.xlsx`: 전처리된 가격 데이터
- `quantity_data_v1.xlsx`: 전처리된 반입량 데이터
